Notifications

notifications#show - Receive your latest notifications

Parameters

Name Description
per_page Filter parameter: define the amount of records sent per call
search[type] Type of reference object accepted, defaults to 'Post', use 'all' for every notification type

Request

Route

GET /v1/notifications

Headers

Accept: application/json
Content-Type: application/json
Authorization: Bearer your_access_token
Host: api.producthunt.com

200 OK
